options

Full name: ase2sprkkr.common.options

Module class hierarchy


Description

The classes for storing one configuration value.

Classes

BaseOption(definition[, container])

A base placeholder for a leaf element of a grammar file, both the a-value-holding ones (Option) and dummy ones (Dummy)

CustomOption(definition[, container, value])

An user-added option (configuration value).

DangerousValue(value[, value_type, validate])

This class is used to store (encapsulate) a value, which should not be validated - to overcame sometimes too strict enforment of the options values.

Dummy(definition[, container])

Option(definition[, container, value])

Class for one option (a configuration value) of SPRKKR - either to be used as a part of InputParameters or Potential configuration.
